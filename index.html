Навигация из продажи → пошагово

Во всех кейсах: Шаг 1 фиксируем статус (бронь/предоплата/оплата) и заполняем форму в Чате оплат. Шаг 2 — «последний чек → полная» + профильные группы.

1) Курс / мини‑курс, базовый (без Zoom и дипломов)
Шаг 1 Статусы: бронь → предоплата → оплата (любая оплата)
  • Сформировать отчет по шаблону → Отправить в чат оплат
  • Заполнить форму по оплате → Ссылка в чате оплат
2) Онлайн‑поток продан
Шаг Статусы: бронь → предоплата → оплата → последний чек → полная оплата (все оплпты)
  • Сформировать отчет по шаблону → Отправить в чат оплат
  • Заполнить форму по оплате → Ссылка в чате оплат
  • Из Чата оплат переслать отчет в группу → Онлайн‑обучение
3) Оффлайн продан
Шаг Статусы: бронь → предоплата → оплата → последний чек → полная оплата (все оплпты)
